# Zoomin Software, Inc.

[Zoomin Software][] has been [acquired by Salesforce][] in autumn 2024.

This Github Org hosts all the main repositories of legacy Zoomin products and associated infra.
It has been migrated under the umbrella Salesforce-Internal [enterprise][] in summer 2025.

New developments of Zoomin (EK / Enterprise Knowledge products) are progressing already integrated
within SFDC change-management workflows -- i.e. on `git.soma`. This org hosts pre-acquisition code.

For further questions, visit the channel [#zoomin-engineering][].
